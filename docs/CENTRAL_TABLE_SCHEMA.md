## Колони (A–AE)

### 🔵 ИДЕНТИФИКАЦИЯ (A–B)
| Кол. | Наименование | Тип | Описание |
|------|-------------|-----|---------|
| A | EAN AMAZON | Number | EAN баркод, видим в Amazon |
| B | EAN Source | Number | EAN баркод от доставчика |

### 🟡 КОРЕКЦИЯ (C–D)
| Кол. | Наименование | Тип | Описание |
|------|-------------|-----|---------|
| C | Корекция / Забележка | String | Бележка за корекция (напр. PACK X 6, правилен ASIN) — **жълт фон** ако е попълнено |
| D | Статус корекция | String | Допълнителен статус |

### 🟢 ДОСТАВЧИК (E–G)
| Кол. | Наименование | Тип | Описание |
|------|-------------|-----|---------|
| E | OUR SKU | String | Наш вътрешен SKU (формат: `ORB-1234567890`) |
| F | SOURCE Catalog Nr. | Number | Каталожен номер при доставчика |
| G | SOURCE | String | Доставчик (Orbico, Akson, Amperel, Fortuna, Uvex...) |

### 🟩 ПРОДУКТ (H–L)
| Кол. | Наименование | Тип | Описание |
|------|-------------|-----|---------|
| H | Brand | String | Марка |
| I | Model | String | Пълно наименование на продукта |
| J | Electronic? | String | Yes / No |
| K | Product (Amazon Link) | URL | Линк към Amazon листинга |
| L | ASIN | String | Amazon Standard Identification Number |

### 🟠 ЦЕНИ AMAZON (M–O)
| Кол. | Наименование | Тип | Описание |
|------|-------------|-----|---------|
| M | Price Competitor EUR Brutto | Decimal | Цена на конкурент в Amazon |
| N | Price Amazon EUR Brutto | Decimal | Текуща цена в Amazon |
| O | Продажна Цена Amazon EUR Brutto | Decimal | **Нашата продажна цена** |

### 🔶 НАШИ ЦЕНИ (P–R) — ФОРМУЛИ
| Кол. | Наименование | Формула | Описание |
|------|-------------|---------|---------|
| P | OUR Price No VAT | `=O/1.19` | Продажна цена без ДДС |
| Q | OUR Price VAT | `=O-P` | Размер на ДДС |
| R | Amazon Fees | `=O*0.15` | Amazon комисионна (15%) |

### 🔴 РАЗХОДИ (S–X) — СМЕСЕНИ
| Кол. | Наименование | Тип/Формула | Описание |
|------|-------------|------------|---------|
| S | Цена Доставчик EUR Netto | Decimal | Наша покупна цена от доставчика |
| T | ДДС от Цена Доставчик EUR | `=S*0.2` | ДДС 20% върху покупната цена |
| U | Транспорт от Доставчик до нас | Decimal | Входящ транспорт |
| V | Транспорт до кр. клиент EUR Netto | Decimal | Изходящ транспорт (доставка до клиента) |
| W | ДДС Транспорт до кр. клиент | `=V*0.2` | ДДС върху изходящ транспорт |
| X | **Result (Марж)** | `=O-Q-R-S-U-V+T+W` | **Реален марж в €** — цветова индикация |

#### Цветова индикация на Result (X):
| Стойност | Цвят | Значение |
|----------|------|---------|
| < 1.00 € | 🔴 Червено | Загуба |
| 1.00–2.00 € | 🟡 Жълто | Нисък марж |
| > 2.00 € | 🟢 Зелено | Добър марж |

### 🟣 ДОПЪЛНИТЕЛНО (Y–AB)
| Кол. | Наименование | Тип/Формула | Описание |
|------|-------------|------------|---------|
| Y | Намерена 2ра обява | String | ASIN на втора обява (ако има) |
| Z | Цена ES/FR/IT | `=O+1.5` | Цена за Испания / Франция / Италия (+1.50€) |
| AA | DM цена | Decimal | Цена в DM канал |
| AB | Нова цена след намаление | Decimal | Временна намалена цена |

### ⚫ СТАТУС (AC–AE)
| Кол. | Наименование | Стойности | Описание |
|------|-------------|----------|---------|
| AC | Доставени | Number | Количество доставени бройки |
| AD | За следваща поръчка | Number | Планирано количество |
| AE | Upload Status | NOT_UPLOADED / UPLOADED / PARTIAL | Статус на качването в Amazon |

## Формули — обобщение

```
P = O / 1.19                          (Цена без ДДС 19%)
Q = O - P                             (Сума ДДС)
R = O * 0.15                          (Amazon комисионна 15%)
T = S * 0.20                          (ДДС върху покупната цена 20%)
W = V * 0.20                          (ДДС върху транспорт 20%)
X = O - Q - R - S - U - V + T + W    (Нетен марж)
Z = O + 1.50                          (Цена за ES/FR/IT)
```
