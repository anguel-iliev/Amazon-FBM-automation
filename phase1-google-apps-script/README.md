# Фаза 1 — Организация на документи (Google Apps Script)

## Какво прави?

Скриптът сканира Gmail за имейли с лейбъл `доставчици`, извлича прикачените файлове и ги организира в Google Drive по следната структура:

```
Google Drive/
└── Цени доставчици/
    ├── Доставчик А/
    │   ├── Ценови листи/
    │   │   ├── 2025-01-15_0900_price_list.xlsx
    │   │   └── 2025-01-20_1100_january_prices.xlsx
    │   └── Фактури/
    │       └── 2025-01-15_0900_invoice_001.pdf
    └── Доставчик Б/
        └── ...
```

---

## Инсталация (стъпка по стъпка)

### Стъпка 1: Създай Gmail лейбъл
1. Отвори Gmail
2. В левия панел → `More` → `Create new label`
3. Въведи: `доставчици`
4. Приложи лейбъла на всички имейли от доставчици

### Стъпка 2: Създай Apps Script проект
1. Отвори [script.google.com](https://script.google.com)
2. Натисни `+ New project`
3. Преименувай на: `Amazon FBM - Phase 1`

### Стъпка 3: Postavi кода
1. Изтрий съдържанието на `Code.gs`
2. Копирай целия код от `Code.gs` в това repo
3. Попълни `CONFIG` обекта:

```javascript
const CONFIG = {
  GMAIL_LABEL: 'доставчици',        // ← лейбъла от Стъпка 1
  DRIVE_ROOT_FOLDER: 'Цени доставчици',
  LOG_SHEET_ID: 'ТВОЯ_SHEET_ID',    // ← от URL на Google Sheet за логове
  // ... останалите са ОК по подразбиране
};
```

### Стъпка 4: Дай permissions
1. В горното меню → `Run` → избери `testRun`
2. Ще те помоли за permissions — натисни `Review permissions` → `Allow`
3. Провери лога (View → Execution log)

### Стъпка 5: Настрой автоматичен trigger
1. Изпълни функцията `setupTrigger` (само веднъж!)
2. Провери: `Extensions` → `Apps Script` → иконата с часовник (Triggers)
3. Трябва да виждаш trigger за `processSupplierEmails` — Daily

## Конфигурация

| Параметър | Описание | Пример |
|-----------|----------|--------|
| `GMAIL_LABEL` | Gmail лейбъл за сканиране | `доставчици` |
| `DRIVE_ROOT_FOLDER` | Root папка в Drive | `Цени доставчици` |
| `LOG_SHEET_ID` | ID на Google Sheet за логове | `1BxiMVs0XRA...` |
| `PRICE_SUBFOLDER` | Подпапка за ценови листи | `Ценови листи` |
| `INVOICE_SUBFOLDER` | Подпапка за фактури | `Фактури` |

## Как се извлича името на доставчика?

Скриптът опитва по следния ред:
1. **Display name** от подателя: `"Иван Иванов <ivan@company.bg>"` → `Иван Иванов`
2. **Домейн** от имейл адреса: `ivan@acme.com` → `acme`
3. **Тема** на имейла (последен вариант)

> 💡 **Препоръка:** Задай правило в Gmail да добавя лейбъла автоматично при получаване на имейл от конкретни адреси.

## Отстраняване на проблеми

| Проблем | Решение |
|---------|---------|
| `Gmail лейбъл не е намерен` | Създай лейбъл `доставчици` в Gmail (lowercase!) |
| `Permission denied` | Изпълни `testRun` отново и allow permissions |
| Файловете не се разпознават | Провери `PRICE_LIST_EXTENSIONS` в CONFIG |
| Дублирани файлове | Нормално — скриптът проверява за дубликати по timestamp |

## Логове

Всяка операция се записва в:
- **Google Apps Script лог:** View → Execution log (само последните изпълнения)
